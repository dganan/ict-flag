Upload instructions:
  • The uploaded file must contain one assignment assessment data on each row.
  • Each row contains the following mandatory fields in this order, separated by commas or semicolons:
    • Code of the tool
    • Unique key of the tool
    • Student username or email
    • Name of the assignment
    • Grade of the assignment (in a range of 0 to 10)
  • Additionaly a row may contain the following optional fields (all three) in this order, separated by commas or semicolons:
    • Code of the Program (as reflected in ICT-FLAG)
    • Code of the Subject (as reflected in ICT-FLAG)
    • Code of the Semester (in the format YYYY-S where YYYY means the year and S is 1 or 2 representing Autumn and Spring semester respectively)
